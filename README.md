# ИП Платежи

Современное веб-приложение для индивидуальных предпринимателей на УСН, которое автоматически рассчитывает предстоящие платежи, налоги, взносы и сроки отчётности.

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшена
npm run preview
```

## Технологический стек

- React 18 + TypeScript
- Vite
- Tailwind CSS v4 + shadcn/ui
- Dexie.js (IndexedDB) для хранения данных
- decimal.js для точных денежных расчётов
- Recharts для графиков
- jsPDF + html2canvas для экспорта PDF

## Основные возможности

### Расчёт налогов
- **УСН «Доходы»** — ставка 6%, уменьшение на взносы (100% без сотрудников, 50% с сотрудниками)
- **УСН «Доходы минус расходы»** — ставка 15%, минимальный налог 1%
- **Авансовые платежи** — автоматический расчёт по кварталам
- **Фиксированные взносы ИП** — 57 390 ₽ (2026)
- **Дополнительный 1%** — с дохода свыше 300 000 ₽
- **НДФЛ** — прогрессивная шкала 13/15/18/20/22%
- **Страховые взносы за сотрудников** — 30% до базы, 15.1% сверх
- **Взносы на травматизм** — настраиваемый тариф
- **НДС при УСН** — опциональный модуль (стандартный 22/10/0%, специальный 5%/7%)

### Функционал
- 6-шаговый онбординг
- Dashboard с карточками и графиком
- Управление доходами и расходами (таблица, фильтры, CSV импорт/экспорт)
- Управление сотрудниками (добавление, редактирование, архивирование)
- Начисление зарплаты (массовое и по одному)
- Календарь платежей и отчётов
- ЕНС (единый налоговый счёт)
- Отчёты и экспорт (CSV, PDF)
- Настройки налоговых правил (редактируемые ставки, лимиты, сроки)
- Праздничные дни с переносом сроков
- Ручные корректировки с журналом изменений
- Тёмная тема
- Адаптивная верстка

## Где менять налоговые правила

1. Перейдите в раздел **Настройки** → **Налоговые правила**
2. Измените нужные ставки, лимиты или тарифы
3. Нажмите **Сохранить**
4. Перейдите в **Расчёт налогов** и нажмите **Пересчитать**

Праздничные дни редактируются в разделе **Настройки** → **Праздники**.

## Ограничения MVP

- Данные хранятся только локально в браузере (IndexedDB)
- Нет подключения к ФНС, банкам, ЭДО
- Приложение не отправляет декларации и отчёты
- НДФЛ для нерезидентов — только как предупреждение
- Нет автоматического расчёта больничных и отпускных
- Нет интеграции с 1С и другими бухгалтерскими системами

## Важное предупреждение

**Приложение не является заменой бухгалтера и не отправляет отчётность в ФНС/СФР.**

Все расчёты носят информационный характер. Перед подачей отчётов и уплатой налогов рекомендуется проверить расчёты с квалифицированным бухгалтером.
